#  E-Commerce Management System

##  Overview
[Project Name] is a comprehensive, full-stack e-commerce web application designed to handle everything from product browsing to secure checkout and backend inventory management. Built using C# and .NET, the platform emphasizes clean architecture, security, and a highly responsive user experience. 

It is divided into two primary environments: a sleek, customer-facing storefront and a powerful, feature-rich administrative dashboard.

##  Key Features

###  Administrator Portal
* **Inventory Management:** Add, edit, and delete products, including uploading product images, setting prices, and managing stock quantities.
* **Category Control:** Create and organize dynamic product categories to keep the storefront highly navigable.
* **Order Fulfillment:** Track customer orders through their entire lifecycle—updating statuses from "Pending" to "Processing," "Shipped," and "Delivered."
* **Customer Directory:** View a complete list of registered users and monitor customer purchasing behavior.
* **Live Support System:** Integrated real-time chat functionality to answer customer questions and provide immediate assistance directly from the admin panel.

###  Customer Experience
* **Secure Authentication:** Full user registration, login, and secure session management.
* **Intuitive Shopping Cart:** Easily browse the catalog, add items to the cart, and view real-time order summaries including VAT calculations.
* **Stripe Integration:** Fast, secure, and reliable payment processing handled via the Stripe API.
* **Transparent Tracking:** Customers can log into their dashboard to view their order history and track the shipping status of their purchases in real-time.
* **Direct Communication:** Built-in live chat allows customers to reach out to store administrators instantly.

##  Technology Stack
* **Backend:** C# .NET 
* **Architecture:** Clean Architecture / MVC Structure
* **Database:** Entity Framework Core 
* **Payments:** Stripe API integration for secure checkout
* **Frontend:** Responsive HTML, CSS, JavaScript
